sdRacer - Info

sdRacer is een spel waarin je code schrijft om een auto te besturen. De auto moet de finishlijn bereiken zonder te crashen.

De auto kan verschillende bewegingen uitvoeren, zoals vooruit rijden, achteruit rijden, en draaien.

Schrijf jouw code in het codeveld, druk op "Check code" om je code te controleren en druk op "Play" om de code uit te voeren.

Je kunt je code opslaan en laden met de "Save" en "Load" knoppen.

Je kunt je code resetten met de "Reset" knop.

Beschikbare commando's:

  • moveForward() - rijdt vooruit
  • moveBackward() - rijdt achteruit
  • turnRight() - draait de auto naar rechts met 90 graden.
  • turnLeft() - draait de auto naar links met 90 graden.
  • canMove() - controleert of de auto vooruit kan bewegen
  • if (condition) { ... } - voert code uit als de conditie waar is
  • while (condition) { ... } - voert code uit zolang de conditie waar is
  • for (int i = 0; i < 10; i = i + 1) { ... } - voert code uit 10 keer
